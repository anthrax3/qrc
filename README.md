What's this?
---------------------------------------------------------------------

This program generates QR code in ASCII Art or Sixel format
for terminals.

Install
---------------------------------------------------------------------

```console
$ go get https://GitHub.com/fumiyas/qrc
```

Usage
---------------------------------------------------------------------

```console
$ $GOPATH/bin/qrc input text
```
